# Fala Oral — Notebook Template Export

> **Duração:** ~1 minuto

---

"Para manter aderência ao template da Trilha I, preparei esta versão demonstrativa do notebook com uma célula final de exportação para PDF via nbconvert e classe article, como no modelo do professor."

"Esse PDF demonstra que o notebook é funcional e exportável no Colab: ele gera automaticamente um arquivo LaTeX intermediário, injeta pacotes de formatação em português, e compila com xelatex e bibtex — exatamente como o template faz."

"Em paralelo, mantive o PDF formal em abnTeX2 como documento de submissão, porque ele oferece melhor acabamento acadêmico: capa normalizada, sumário automático, citações ABNT, e estrutura de 72 páginas."

"Portanto, o notebook demonstra reprodutibilidade; o PDF abnTeX2 consolida a versão formal. São complementares, não contraditórios."

---

## Se a banca perguntar: "Qual PDF é o oficial?"

> "O PDF abnTeX2 de 72 páginas é o documento formal. O PDF do notebook é demonstrativo — mostra que o notebook funciona e gera saída equivalente ao template."

## Se a banca perguntar: "Por que dois PDFs?"

> "O template gera PDF com classe article, que é funcional mas não tem acabamento abnTeX2. Para a submissão formal, uso abnTeX2. Para demonstração de reprodutibilidade no Colab, uso o mesmo padrão do template."
